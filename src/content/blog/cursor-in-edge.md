엣지, 크롬등 블링크 엔진 기반 브라우저에서 텍스트 위에 마우스 포인터(커서)를 갖다대었을 때, 'I' 모양의 커서나 나타나지 않고 사라지는 증상이 있음. 케이스에 따라 경우는 약간씩 달라지는데 마우스 포인터의 색상이 흰색으로 바뀌면서 잘 보이지 않는 경우부터 아예 사라지는 경우까지. 

몇 가지 동작을 수행하면 다시 보이기도 하는데, 곧 증상이 재발함. 

1. 마우스 포인터 속도 조절
2. TEXTAREA 영역을 클릭하여 글자 입력이 가능하도록 함

근본적인 원인은 브라우저 렌더링 엔진 자체의 버그로 보이며, 하드웨어 가속을 끄고서 사용하면 증상이 나타나지 않는 것으로 보고됨. 그러나 하드웨어 가속을 끄면 화면 스크롤 등에서 직접적인 사용자 경험 저하를 체감할 수 있어서 권장되지는 않음.

다른 방안으로 마우스 포인터 모양을 커스텀하면 증상이 나타나지 않음. 


1. 제어판 열기
2. 마우스 진입
3. 포인터 탭
4. 사용자 지정 항목 중 '텍스트 선택'을 선택하고, [찾아보기...] 버튼 클릭.
5. `beam_r.cur` 파일을 선택
6. 확인 버튼을 클릭하여 설정 저장.
