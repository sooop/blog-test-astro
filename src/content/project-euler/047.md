소인수 분해 함수는 이전에 작성한 것이 있으니 그것을 사용하면 됩니다. 

```python
from euler import timeit, factorize

@timeit
def main():
    n = 2 * 3 * 5 * 7
    j = 0
    while j < 4:
        j = (j + 1) if len(factorize(n)) == 4 else 0
        n += 1
    print(n-4, factorize(n-4))

main()


if __name__ == "__main__":
    main()
```
