## 풀이

문제를 잘 읽어보면 아주 간단한 내용이라는 것을 알 수 있습니다.

```python
def test(n: int) -> bool:
    for _ in range(50):
        n += int(str(n)[::-1])
        if str(n) == str(n)[::-1]:
            return False
    return True


def main():
    print(sum(1 for n in range(10_000) if test(n + 1)))


if __name__ == '__main__':
    main()
```
